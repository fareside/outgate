# Contributing

Bug reports and pull requests are welcome on GitHub at https://github.com/fareside/outgate.

## Getting started

```bash
git clone https://github.com/fareside/outgate
cd outgate
cargo build
```

See [Building from source](docs/development.md) for the full development setup.

## Guidelines

- Open an issue before starting significant work so we can discuss the approach.
- Keep pull requests focused — one thing per PR.
- Existing tests should pass; new behaviour should have tests.
